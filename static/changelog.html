microformat-shiv

A cross browser javascript microformats parser

Change logs

The history of the project's development is covered in more detail in the releases information on the GitHub project. Below is an overview:

Historic change logs

1.3.2 - 28 Dec 2015

  • Updated URL parsing module

1.3.1 - 8 Oct 2015

  • Added modern version of code without polyfills

1.3.0 - 25 Sept 2015

  • Removed 'alternate' structures as per specification change Sept 2015

1.2.0 - 9 Sept 2015

  • Updated backward compatibility parsing. Now blocks overlapping properties from different microformat versions and sets implied properties by microformat version

1.1.4 - 8 Sept 2015

  • Added options.html support for get and count

1.1.1 - 21 Aug 2015

  • Upgraded backwards compatible parsing options

1.1.0 - 20 Aug 2015

  • Added support for v1 title to job-title

1.0.2 - 8 Aug 2015

  • Updated test structure for Firefox and browser support info in readme.md

1.0.0 - 27 July 2015

  • Updated to latest tests and made ready for use in Firefox

0.3.1 - 6 July 2013

  • Added relative URL expandition within HTML

0.3.0 - 22 Novemeber 2012

  • Full rewrite of code to provide microformats 2 parser

0.2.4 - 22 Novemeber 2011

  • Merge of Fabian Beiner changes. Last commit before full rewrite. SHA a8bab70a758bcf76e493a9b007726e9a5c1b170f

0.2.4 - 18 Novemeber 2011

  • Pull request - Checks if given element is already a vcard element. (Fabian Beiner)

0.2.3 - 4 June 2011

0.2.2 - 26 May 2011

0.2.1 - 18 May 2011

  • A small speed increase

  • Corrected a number of edge case parsing problems

  • Packaged the returned data using ufJSON schema Note: breaks with the original output schema

  • Corrected hCard extended-street property so it returns an array
  • Improved the hResume definition
  • Reorganized the parser code to make it clear what are public and private methods
  • Wrapped all definitions so they were not in the global scope
